Install
------------------------------------------------------------------------------

``learn_redshift`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install learn_redshift

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade learn_redshift
